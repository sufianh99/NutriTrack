    **GREEN phase:** Create `app/nutrition.py` as a pure-function module (NO Flask, SQLAlchemy, or any framework imports). Module docstring: `"""Pure-function nutrition module for portion scaling, summation, and progress status.\n\nNo Flask or SQLAlchemy imports allowed in this module.\n"""`

    Three functions:

    1. `scale_nutrients(amount_g: float, calories_per_100g: float, protein_per_100g: float, fat_per_100g: float, carbs_per_100g: float) -> dict[str, float]`:
       - `factor = amount_g / 100.0`
       - Return `{"calories": round(calories_per_100g * factor, 1), "protein_g": round(protein_per_100g * factor, 1), "fat_g": round(fat_per_100g * factor, 1), "carbs_g": round(carbs_per_100g * factor, 1)}`

    2. `sum_daily_nutrients(entries: list[dict[str, float]]) -> dict[str, float]`:
       - Return `{"calories": round(sum(e["calories"] for e in entries), 1), "protein_g": round(sum(e["protein_g"] for e in entries), 1), "fat_g": round(sum(e["fat_g"] for e in entries), 1), "carbs_g": round(sum(e["carbs_g"] for e in entries), 1)}`
       - Empty list returns all zeros (sum of empty iterable is 0)

    3. `progress_status(actual: float, goal: float) -> str`:
       - If `goal <= 0`: return `""`
       - `ratio = actual / goal`
       - If `ratio < 0.90`: return `""`
       - If `ratio <= 1.00`: return `"success"`
       - Else: return `"danger"`
       - Thresholds: 0.90 and 1.00 per ROADMAP.md success criteria (NOT 0.95/1.05 from old ARCHITECTURE.md)
