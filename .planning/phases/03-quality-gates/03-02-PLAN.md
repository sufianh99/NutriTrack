- Assert saved profile.age == 30, profile.weight_kg == 70.0, profile.gender == "male"

**test_food_entry_add(client, app):**

- Assert response is redirect (status 302)
- Within app_context, query all FoodEntry rows and assert count == 1
- Assert the entry.name == "Banane" and entry.amount_g == 120.0

**test_dashboard_response(client, app):**

- Then GET `/dashboard`

- Assert `b"Protein"` in response.data (dashboard shows macro info)
